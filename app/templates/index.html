Email.Cloud Queue Management

This application allows you to manage the queue of Email.Cloud tenant.

Installation Email.Cloud API user Quick start

There are two ways how you can run the EQAPI application:

  • Build and run a docker container (recommended)
  • Manually run app with python flask
Docker Manual

Prerequisites

Docker must be installed on the computer or server on which you will execute this procedure. This procedure can be executed on a desktop with Docker CE or on a linux server that is running Docker. Please find more information how to install docker at https://docs.docker.com/engine/install/.

# Description
1

Visit the Github repository which holds the source code of Eqapi, click on the Code dropdown button and select Download ZIP.
2

Extract the ZIP file and open a command prompt in the directory that contains the extracted content from the ZIP file.
3

Build the docker container with the following command:

docker build -t eqapi .

Docker starts building the container, running through all the steps and ends with the build tag, which should be eqapi:latest.

Verify that the docker image is added to your local repository with the command:

docker image ls eqapi

4

Create a database folder that will hold the database with the command:

mkdir db
5

Start the docker container with the following command:

docker run --name eqapi -d -p 8000:5000 -v db:/home/eqapi/db eqapi

Verifiy that the container has been created with the following command:

docker ps

CONTAINER ID   IMAGE     COMMAND       CREATED          STATUS          PORTS                    NAMES

6

Open a browser and connect to https://localhost:8000/. It should load the EQAPI web application. Proceed to the login page where you can register your user account and start using the application. Be sure to create a new Email.Cloud API user as outlined above.
7

From now on the docker container can be started (after a reboot for example) with this command: docker start eqapi.

Prerequisites

Python 3.8 must be installed on the system you are executing this procedure.

# Description
1

Visit the Github repository which holds the source code of Eqapi, click on the Code dropdown button and select Download ZIP.
2

Extract the ZIP file and open a command prompt in the directory that contains the extracted content from the ZIP file.
3

Create a new virtual environment with the following command:

python -m venv venv

Activate the virtual environment

  • Linux: source venv/bin/activate
  • Windows (CMD): venv\Scripts\activate.bat
4

Install the requirements with the following command:

pip install -r requirements.txt
5

Create the database folder with following command:

mkdir db

and create the database with:

flask db upgrade

6

Run the app with the following command:

flask run

The application should now be accesible on https://localhost:5000/

To be able to connect to the queue management API we need to supply an username and password with the correct privileges. We recommend that you create a new user for the datafeed API.
